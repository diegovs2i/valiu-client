## Getting Started

Install (preferred=Node.JS 20) and then run the app
| Backend is on valiu-app repo and also should be running please check [Valiu-app](https://github.com/diegovs2i/valiu-app)

## Installation

```bash
$ npm install
```

## Running the app

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```
